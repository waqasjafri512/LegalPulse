# Security Policy

## Supported Versions

Only the latest version of LegalPulse is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of LegalPulse seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it privately.

Please send an email to **security@legalpulse.ai** (placeholder) with the following information:

* Type of issue (e.g., buffer overflow, SQL injection, RCE)
* Full paths to source file(s) related to the manifestation of the issue
* The location/URL of the affected endpoint (if applicable)
* A proof-of-concept or exploit code
* Impact of the vulnerability

We will acknowledge your report within 48 hours and provide a timeline for a fix.
